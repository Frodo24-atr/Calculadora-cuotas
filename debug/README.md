# 🐛 Debug Files

Esta carpeta contiene archivos utilizados para depuración y diagnóstico del proyecto.

## Estructura

### `/html/`
Archivos HTML de depuración específicos:
- `debug.html` - Debug general del sistema
- `debug_boton_completo.html` - Debug específico de botones
- `debug_eliminar.html` - Debug de funcionalidad de eliminación
- `debug_grafico.html` - Debug del sistema de gráficos

### Archivos de Script
- `check_ids.sh` - Script para verificar IDs duplicados o faltantes

## Uso

Estos archivos se utilizan durante el desarrollo para:
1. Identificar y resolver problemas
2. Verificar el comportamiento de componentes específicos
3. Probar correcciones antes de aplicarlas al código principal

## Nota

Estos archivos no son necesarios para el funcionamiento en producción y pueden ser eliminados una vez que el proyecto esté estable.
